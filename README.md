# AskSeniors - A College Q&A Platform

AskSeniors is a web platform designed to connect college juniors with seniors to ask questions, share placement experiences, and receive peer-to-peer guidance—all within a verified student community.

## 🔍 Project Overview

AskSeniors helps students:
- Ask questions related to academics, placements, internships, and campus life.
- Get answers from verified seniors of the same college.
- View real experiences, interview processes, and tips from seniors.

## 💡 Key Features

- 🎓 **College Email Verification:** Only students with valid college email IDs can register.
- 🗨️ **Ask Questions:** Easily post questions to the community.
- ✅ **Answer & Upvote:** Seniors can answer; users can upvote helpful answers.
- 🧑‍🤝‍🧑 **Experience Sharing:** Share and read placement stories or interview tips.
- 🔎 **Search Functionality:** Search by topics, companies, or departments.
